Testing Checklist

Use this checklist to verify all functionality works correctly:

  • □ Start a typing session by selecting text
  • □ Verify hints panel appears with WPM, accuracy, and shortcuts
  • □ Click X button on hints panel - should hide the panel
  • □ Right-click extension button - context menu should appear
  • □ Use "Show Hints Panel" from context menu - should show hints
  • □ Use "Hide Hints Panel" from context menu - should hide hints
  • □ Open extension popup and test show/hide buttons
  • □ Toggle hints visibility checkbox in popup
  • □ End session and start new one - hints visibility should persist
  • □ Test on different websites to ensure consistency

Expected Behavior

  • Hints Panel: Should show current WPM, accuracy, time, and keyboard shortcuts
  • X Button: Should hide hints panel and save preference
  • Context Menu: Should provide options to show/hide hints
  • Popup Controls: Should allow toggling hints visibility
  • Settings Persistence: Hint visibility preference should be remembered
  • Error Handling: Should show appropriate messages for invalid actions
